# Design Spec — Owners Strategy

## Design Tokens (Global)

Apply these tokens to every chart, dashboard, visualization, and designed artifact produced from this repo.

| Token | Value | Use |
|---|---|---|
| `--near-black` | `#0E1821` | Primary dark background |
| `--deep-slate` | `#3E5268` | Secondary dark, panels |
| `--parchment` | `#EDE8DF` | Warm light background |
| `--parchment-2` | `#E4E8EE` | Cool light background |
| `--orange` | `#FF5200` | CTAs, kickers, accents — use sparingly |
| `--tan` | `#B5C4D4` | Secondary emphasis in headlines |
| `--white` | `#FFFFFF` | Body text on dark backgrounds |
| `--f-display` | `'Inter Tight'` | Headlines, stats, large numbers |
| `--f-body` | `'Inter Tight'` | Body copy, labels, UI text |

**Orange (`#FF5200`) is the single accent color.** Use it only for CTAs, kicker labels, left borders on quote cards, and `<em>` highlights. Never decoratively.
